# 💰 Keuangan Pribadi - Google Sheets

Sistem pencatatan keuangan pribadi berbasis Google Sheets + Apps Script.

## 🚀 Fitur
- Input transaksi (Income & Expense)
- Kategori pengeluaran
- Dashboard otomatis
- Laporan bulanan
- Saldo real-time

## 🧱 Struktur
- M_AKUN
- M_KATEGORI
- TRANSAKSI
- LAPORAN
- DASHBOARD

## ⚙️ Teknologi
- Google Sheets
- Google Apps Script (JavaScript)

## 📌 Tujuan
Membantu pencatatan dan kontrol keuangan pribadi secara sederhana namun powerful.
